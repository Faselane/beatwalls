# <init>

Define()

Define your own WallStructure from existing WallStructures.

Define your own structure with something like this

myLine: Line
 p1: 4,0,0
 p2: 2,4,2

and then use it with

10: myLine

You can also supply more then one structure. For example when you have more then one line already defined

myLine:
 structures: _line1, _line2
